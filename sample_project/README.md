# Calculator Project

This sample project is used to demonstrate Git and GitHub workflows.

## Features

- Addition
- Subtraction
- Multiplication
- Division

## Learning Objectives

Students will use this project to learn:

- Commits
- Branches
- Merging
- Conflict Resolution
- Pull Requests
- Tags
- Releases

## Development Workflow

Each new feature should be developed in a separate branch and merged through a Pull Request.
